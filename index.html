How to embed other games

  1. Find the folder name of the game you want (for example: 2048).
  2. Build the URL: https://raw.githack.com/3kh0/3kh0-assets/main/<FOLDER>/index.html
  3. Use it in an iframe: <iframe src=".../FOLDER/index.html"></iframe>
